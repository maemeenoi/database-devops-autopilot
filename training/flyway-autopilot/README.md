# Flyway Autopilot Training

Welcome to the Flyway Autopilot training module. This comprehensive workshop will guide you through implementing automated database deployment pipelines using Flyway and modern DevOps practices.

## Learning Objectives

By the end of this training, you will be able to:

- Set up automated database migration pipelines
- Implement version control for database schemas
- Configure monitoring and rollback strategies
- Apply DevOps best practices to database deployments

## Training Labs

1. [Lab 1: Getting Started](labs/lab1-getting-started.md) - Introduction to Flyway Autopilot
2. [Lab 2: Pipelines and Automation](labs/lab2-pipelines-and-automation.md) - Setting up CI/CD pipelines
3. [Lab 3: Deployments](labs/lab3-deployments.md) - Automated deployment strategies
4. [Lab 4: Version Control](labs/lab4-version-control.md) - Database version management
5. [Lab 5: Monitoring and Rollback](labs/lab5-monitoring-and-rollback.md) - Monitoring deployments and rollback procedures
6. [Lab 6: Best Practices](labs/lab6-best-practices.md) - Industry best practices and advanced techniques

## Prerequisites

Before starting this training, ensure you have:

- Basic understanding of databases and SQL
- Familiarity with version control (Git)
- Access to the required development environment
- Completed the [prerequisites checklist](../../docs/prerequisites.md)

## Duration

This training is designed to be completed in approximately 4-6 hours, depending on your experience level.
